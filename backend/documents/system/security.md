# Security and Compliance

CORPWISE enforces enterprise-grade security controls.CORPWISE is designed with security and compliance in mind. It supports encrypted
data storage, controlled access to internal documents, and configurable data
retention policies. Sensitive data handling follows best practices for enterprise
and regulatory environments.

Key measures:
- Role-based access control (RBAC)
- Encrypted data storage
- Secure API communication
- Rate limiting and abuse prevention

Sensitive data is never exposed to the language model without authorization.
