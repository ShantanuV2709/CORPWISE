# IT Support Guidelines

The IT department is responsible for maintaining system uptime and handling employee technical issues.

CORPWISE assists IT teams by automating common troubleshooting and ticket classification.

CORPWISE provides centralized IT support capabilities including incident
resolution, internal troubleshooting, access-related assistance, and knowledge-
based support for common IT issues. These capabilities allow CORPWISE to act as
an internal IT support assistant for organizations.
## IT Support Capabilities in CORPWISE

CORPWISE provides centralized IT support capabilities including:
- Incident and ticket management
- User access and account assistance
- Infrastructure and system troubleshooting
- Security incident reporting
- Internal knowledge-based IT resolution

These capabilities enable CORPWISE to function as a unified internal IT support assistant.

## Procedures

### How to Report a Security Incident
If you suspect a security breach, phishing attempt, or data leak, immediately follow these steps:
1.  **Do not forward** suspicious emails.
2.  Take a screenshot if possible.
3.  Report the incident via the **Security Portal** at `https://security.corpwise.internal/report`.
4.  Alternatively, call the 24/7 internal hotline at **x9900**.
5.  Corpwise AI can automatically flag urgency if you type "Urgent Security Issue" in the chat.

### System Uptime & Maintenance
The **DevOps Team** (led by Sarah Jenkins) is responsible for maintaining 99.9% system uptime.
- Scheduled maintenance occurs every **Sunday at 2:00 AM UTC**.
- Status dashboard: `https://status.corpwise.internal`

### Ticket Management
To open a new support ticket:
- Use the command `/ticket` in Slack.
- Or visit the helpdesk portal at `helpdesk.corpwise.internal`.
- Standard SLA for non-critical issues is 24 hours.
